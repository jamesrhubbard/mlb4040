Architecture

How It Works

01
MLB API Poll
The bot polls the free MLB Stats API continuously during game hours — detecting every home run and stolen base within seconds of it happening on the field.
02
Event Detection
New stats are compared against state saved in Google Cloud Firestore. Any new HR or SB triggers an event. Milestones (10, 20, 30, 35, 38, 39, 40) get special treatment.
03
Monte Carlo
10,000 season simulations run for every player on every poll. Each sim applies 8+ real-world factors to calculate a true probability % — not just pace math.
04
Graphic + Tweet
A custom 1200×1200 PNG is generated with the triggering player's card on top and the full ranked leaderboard below. Auto-posted to @4040tracker instantly.
